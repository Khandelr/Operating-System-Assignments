myshell version 1.0 09/12/2013

GENERAL USAGE NOTES
--------------------

- myshell is our own shell written in C.

- It is a command line interpreter which reads sets of commands typed at prompt and executes them.

- It is like a unix shell which executes commands which the users give on the terminal.

- For example,
		myshell: cat f1 
The above command will display the contents of file 'f1'.
